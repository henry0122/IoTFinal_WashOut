# IoTFinal_WashOut

## Using Sensor:
```
    1. Triaxial acceleration sensor
    2. Light sensor
```

## Goal:
```
    Know whether the washing machine is working, and show by this app.
```

## Method:
```
    1. Sensor data will be sent to Mediatek Cloud Sandbox(MCS) through wi-fi.
    2. App will get the data by using http method.
    3. Then show the result after analyzing the data.
```
